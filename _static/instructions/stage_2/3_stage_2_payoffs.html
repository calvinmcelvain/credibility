There are a number of differences between Stage 1 and Stage 2.

  1. The payoff tables will change for each decision in Stage 2. Be certain to look at the payoff tables and instructions before making your decisions in Stage 2, as these will not be the same as Stage 1 or the other decisions in Stage 2.
  2. Investors’ endowments are not the same for every decision in Stage 2. Instead, for each decision, an endowment will be randomly drawn for each Investor. The next few screens explain exactly how this works and how Investors will choose between “Invest” and “Keep”.
